# Contributor Covenant 2.1

We follow the Contributor Covenant to keep this project welcoming for everyone.

- **Be respectful and professional.** Disagreements happen; keep them about the work.
- **Assume good faith.** Questions and contributions come from people at different skill levels.
- **Focus on the engineering.** This profile's value is its hooks, agents, and
  contracts — critique the design, not the person.
- **No harassment, discrimination, or personal attacks.** This is a hard boundary.

## Enforcement

Report unacceptable behavior to the maintainer (see [`SECURITY.md`](SECURITY.md)
for private contact). Reports are handled privately. Repeated or severe
violations result in contributions being declined and, where possible, the
person being removed from project spaces.

This code is a summary; where it conflicts with the full
[Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
the full text governs.
